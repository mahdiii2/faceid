Processing Rules

  • Only Access Granted by Face events are used
  • Authentications within 30 minutes are merged into one cluster
  • First Register = first cluster start of the day
  • Last Register = last cluster end of the day
  • Lunch break (12:00–13:00) and evening break (17:00–18:00) are deducted
  • Output includes Daily Summary sheet
